# CSS 문법

```css
div {
  font-size: 20px;
  color: red;
}

선택자 {
  속성: 값;
  속성: 값;
}
```

## 선택자

```html
<div>
  <h1>제목</h1>
  <p>본문..</p>
</div>
```

```CSS
h1{
    color:red;
}
p{
    color:blue;
}
```

## 속성(Properties)과 값(Value)

html 속성은 attribute

```css
div {
  color: red;
  font-size: 20px;
  width: 300px;
  margin: 20px;
  padding: 10px 20px;
  position: absolute;
}
```
