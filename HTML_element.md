# 문자 콘텐츠

## <ol>, <ul>, <li>

- <li> : 목록을 만드는 콘텐츠
- <ol> : 정렬된 목록, 순서가 있음
- <ul> : 정렬되지 않은 목록 , 순서가 필요 없음

1. <ol> 과 <ul>은 자식으로 <li>만 포함 가능
2. <li>는 단독으로 사용할 수 없음
3. <ol>의 항목 순서는 중요도를 의미할 수 있음

## <dl>, <dt>, <dd>

용어:<dt> 설명:<dd> 쌍들의 영역: <dl>

- definition term
- definition details
- definition list

* <dl>은 <dd>, <dt>만을 포함해야 한다.
