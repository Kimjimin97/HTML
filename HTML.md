# HTML 문법

## 기본 형태

```html
<TAG>CONTENT</TAG> ##요소
```

```html
<h1>토끼와 거북이</h1>
<p>옛날 옛적에 토끼와 거북이가 살았습니다.</p>
```

## 속성과 값

```html
<TAG 속성="값"></TAG>
```

```html
<img src="./my_photo.jpg" alt="내 프로필 사진" />

<div class="name">홍길동</div>
```

## 부모와 자식 요소

상위, 하위

```html
<PARENT>
  <CHILD></CHILD>
</PARENT>
```

```html
<section class="fruits">
    <h1>과일목록</h1>
    <ul>
        <li>사과</li>
        <li>딸기</li>
        <li>오렌지</li>
    <ul>
</section>
```
